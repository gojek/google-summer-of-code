# Contributor Guidelines for Google Summer of Code 2026

## General guidelines

* Read the [Official Contributor Guide](https://google.github.io/gsocguides/student/) by Google Summer of Code.
* Check out the [GSoC 2026 Project Ideas List](google-summer-of-code-2026.md) and contributing guide available on each individual project repository.
* Join our [Discord server](https://discord.com/invite/C823qK4AK7) and introduce yourself in the [#google-summer-of-code-2026](https://discord.com/channels/986889666651373568/1468219617066356807) channel.
* Ask for any development-related help or GSoC related queries in the above channel.
* Start as early as possible and it is always better to contribute in [Gojek GitHub Org](https://github.com/gojek) before submitting your proposal. This showcases your technical skills and help the mentors make an informed decision. 
* Other form of contributions like helping out the community on Discord, GitHub issues, and reviewing pull requests won't go unnoticed.
* Make sure to reach out and interact with the respective mentors to get their reviews before the final submission.
